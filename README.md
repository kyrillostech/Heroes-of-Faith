**Heroes of Faith** is an interactive web-based Memory Matching game designed specifically for **Coptic Orthodox Sunday Schools**. It combines modern "Gamification" techniques with spiritual heritage to help children learn about Saints, Martyrs, and Holy Symbols in an engaging way.

### ğŸŒŸ Key Features
- **Authentic Iconography:** Features high-quality icons of St. George, St. Mary, Pope Kyrillos VI, and more.
- **Bilingual Interface:** Fully Arabic interface with spiritual encouragement phrases.
- **Responsive Design:** Works perfectly on Mobile, Tablets, and Desktop boards.
- **Offline Capable:** All assets are local, making it ideal for church classrooms without internet.

## ğŸ® How to Play
1. **Choose Level:** Select **Beginner** (6 Pairs) or **Advanced** (8 Pairs).
2. **Flip Cards:** Click on cards to reveal the Saint or Symbol.
3. **Match Pairs:** Find two identical icons to clear them.
